实现手势密码：
1、画出9个密码点，并将9个密码点的位置信息保存起来，以及根据位置查找对应密码点的路径序号存起来
2、用户点击设置密码
3、触发touch事件，包括开始触摸，手势划过事件，以及手势结束事件
4、开始触摸时，将该点加入路径信息中
5、手势划过时，需要先判断该点是否已存在在路径中，如果不存在，则加入到路径信息中，然后根据路径信息中的点划线
6、手势结束时，先要判断路径信息中的点是否小于5，如果小于，则清除画布，以及路径信息，重绘画布，提示用户重新绘制手势，如果大于5，则将路径信息存入localStorage中，并提示用户再次绘制手势，如果与localStorage中存储的信息一致，则提示用户手势设置成功，并清除画布，以及路径信息，否则，删除localStorage中的路径信息，清除画布以及路径信息，提示用户重新绘制手势密码
7、用户点击验证密码
8、触发touch事件，包括开始触摸，手势划过事件，以及手势结束事件
9、开始触摸时，将该点加入路径信息中
10、手势划过时，需要先判断该点是否已存在在路径中，如果不存在，则加入到路径信息中，然后根据路径信息中的点划线
11、手势结束时，需要判断此时的路径信息与localStorage中存储的路径信息是否一致，若一致，则清除画布，以及路径信息，提示用户手势密码绘制正确，否则，清除画布以及路径信息，提示用户密码输入错误，请重新绘制手势密码
